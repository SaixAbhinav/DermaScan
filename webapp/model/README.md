# model/ — trained artifacts

This directory holds the artifacts produced by the **export cell** at the end of
[`../../Skin lesion Analyizer.ipynb`](../../Skin%20lesion%20Analyizer.ipynb).
They are **not committed to git** (see `../.gitignore`) because they are large
and regenerated by training.

After training, copy these files here:

| File | Produced by | Used for |
|------|-------------|----------|
| `model.keras` | training checkpoint | predictions |
| `class_indices.json` | export cell | authoritative class order |
| `metrics.json` | export cell | About page (accuracy, per-class F1) |
| `confusion_matrix.png` | export cell | About page image |
| `examples/<class>.jpg` | export cell | "Try an example" gallery |

To point the app at artifacts stored elsewhere, set the `MODEL_DIR` environment
variable.

The app still **starts without these files** — the frontend loads and the
About page renders, but `/api/predict` returns HTTP 503 until `model.keras`
is present.
